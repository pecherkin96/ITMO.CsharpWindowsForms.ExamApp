Данное решение задания создано при помощи элемента DataGrid.
Подключение к базе данных установлено при помощи элемента DataGrid.

Предустановка базы данных.
1. Скачайте два файла базы данных из этого каталога: test_block.mdf и test_block_log.ldf
2. Развените базу данных на своём локальном сервере.
3. Заустите файл .sln в Visual Studio чтобы настроить подсоеденеие базы данных к DataGrid
4. Нажимаем в верхнем меню Visual Studio "Проект"- "Добавить новый источник данных" выбираем там
развёрнутую базу данных.
5. Собираем решение - запускаем проект.
